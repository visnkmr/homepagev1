Projects

No Keyboard

A revolutionary keyboard that allows you to hide your keyboard when not required. Also useful if you use bluetooth keyboard or just want to scroll through text on device without the keyboard popping up.

AppInfo Manager/ Applist Manager (Teave)

Easily find, search, browse, sort, manage, extract APK of installed User and System apps. Fuzzy logic based search, Uses StorageAccessFramework for saving extracted APK.

Volume Change UI: Software Volume Button

Software Volume Button to open the default Volume Change UI on your device.

File Explorer

A basic file explorer. Implements Data parallelism for select discovery functions, Fuzzy logic based search, stack based history, async based search. Made using Java, Gradle, Android Studio.

Power Menu: Software Power Button

Quickly open the system default Power Menu UI on your device.

Wireless File Manager

Transfer files to your android/Fire devices using any web browser. Amazon Appstore version supports transfers to both Internal Storage, External Storage, A limited iteration is available on Google Play that uses Storage Access Framework and requires no permission but can only send to Downloads folder.

Perlink

Open source utility to choose browser on a per link basis on linux, windows or mac. Made using Rust, VS Code.

Take Notes

Take notes in browser.

Netspeed Test

Test Internet download speed, latency (ping), scan LAN / WiFi for connected devices; all while consuming the minimum amount of data required.

NetSpeed Monitor

Monitor live bandwidth usage/ network speed on any device running on any OS except iOS. Uses peristent XML overlay with Java for android, Rust daemon with a python frontend on PCs.

Taotlus (Launcher)

Taotlus lists all apps installed on your device; including side loaded apps; can be used as a Launcher if required. Implements Fuzzy logic based search, SQL as DB.

Background Apps and Process List

Force close apps that are running in the background. Made using Java, Gradle, Android Studio.
